## Flagship Project

<details>
<summary><b>Fyrex — Modular Discord Bot for Minecraft Communities</b></summary>
<br/>

Fyrex is a primary software project — a modular Discord bot built for Minecraft communities. It focuses on moderation, automation, invite tracking, ticket management, logging, staff tools, and scalable server management. Designed with an extensible architecture that makes adding new features straightforward while maintaining performance and code quality.

| Aspect | Details |
|---|---|
| Stack | Node.js, Discord.js, Express.js, Prisma ORM, SQLite/MySQL |
| Scale | Multi-guild, modular command and event architecture |
| Performance | Optimized event handling and database queries |
| Security | Role-based permission gating, input validation, rate limiting |
| Impact | Streamlines moderation, automation, and staff workflows for Minecraft communities |
| Repository | [github.com/auradad](https://github.com/auradad) |
